# HealthHunt ❤️

The best of health, all in one place.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)

### Installation

1. Clone the repo

  ```sh
  git clone
  ```

2. Install packages

  ```sh
  pnpm install
  ```

## Usage

```sh
pnpm run dev
```
